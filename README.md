## ğŸ§ª How to Run Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/expense-tracker.git
   cd expense-tracker
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   ```

   ```

4. **Run the app**

   ```bash
   npm run dev
   ```
